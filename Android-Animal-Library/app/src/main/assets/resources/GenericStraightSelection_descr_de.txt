Die Lineare Suche ist wohl das trivialste Suchverfahren.

Angefangen mit dem ersten Element werden die gegebenen Daten
- hier ein Array - der Reihe nach überprüft, ob sie dem gesuchten
Element entsprechen. Falls dies der Fall ist, wird die aktuelle
Position als Ergebniswert zurückgegeben. Andernfalls wird die
Suche mit der nächsten Position fortgesetzt.

Wird das gesuchte Element nicht gefunden, wird der Wert -1 als
'Fehlermarkierung' zurückgegeben.
